Une boîte verte

les boîtes de type bloc "block" sont affichées dans une succession verticale. Eléments créant des boîtes bloc sont :

  • élément div
  • titres h1, h2, h3, h4, h5, h6
  • paragraphe p
  • listes et éléments de liste ul, ol, li, dl, dd
  • bloc de citation blockquote
  • texte pré-formaté pre
  • adresse address

les boîtes de type en ligne "inline" sont affichées dans une succession horizontale avec séparation <span. Eléments créant des boîtes en ligne sont :

  • élément span
  • lien a
  • image img
  • emphases em, strong
  • citations q
  • texte entré par l'utilisateur kbd
  • exemple samp
  • variable var
  • abréviations et acronymes abbr, acronym
  • texte inséré ins
  • texte supprimé del
